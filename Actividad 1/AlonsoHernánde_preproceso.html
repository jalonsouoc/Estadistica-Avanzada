9.1. Funciones de medida robustas

Para obtener medidas robustas de nuestros datos en pirmer lugar definimos una funciÃ³n que permita calcular la media recortada, dicha funciÃ³n se va a encargar de ordenar el vector de datos y posteriormente a eliminar los elementos necesarios al inicio y final de dicho vector para el calculo de la media:

media.recortada <- function(x, perc=0.05 ){
 
  # Ordenamos el vector
  x <- sort(x, decreasing = FALSE, index.return = FALSE)
  # Obtenemos el nÃºmero de elementos 
  n <- length(x)
 
  # Obtenemso el nÃºmero de elementos a eliminar al principio y al final del vector
  nValoresExluidos <- round(n * perc)
  indiceSuperior <- n - nValoresExluidos
  
  # Eliminamos del vector los primeros y Ãºltimos nValoresExcluidos
  x <-  x[nValoresExluidos:indiceSuperior]
  
 return(mean(x))
}

Por otro lado definimos una funciÃ³n que permita obtener la media winsor, dicha funciÃ³n se encargara de ordenar el vector y posteriormente sustituirÃ¡ los elementos iniciales y finales por el valor de corte en Ã¡mbas partes del vector para el calculo de la media

media.winsor <- function(x, perc=0.05){

    # Ordenamos el vector
  x <- sort(x, decreasing = FALSE, index.return = FALSE)
  # Obtenemos el nÃºmero de elementos 
  n <- length(x)
 
    # Obtenemso el nÃºmero de elementos a sustituir al principio y al final del vector
  nValoresSustituir <- round(n * perc)
  
  #Obtenemos los valores por los que vamos a sustituir los nValoresSustutir al inicio y al final
  valorInferior <- x[nValoresSustituir + 1]
  valorSuperior <- x[n - nValoresSustituir - 1]
  indiceSuperior <- n - nValoresSustituir
  
  #SUstituimos los elementos
  for(i in 1:nValoresSustituir){
    x[i] <- valorInferior
  }
  
  for(i in indiceSuperior:n ){
    x[i] <- valorSuperior
  }
  
 return(mean(x))
}
